📋 ФАЙЛЫ ДЛЯ ЗАПУСКА:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. start-all.bat
   🚀 РЕКОМЕНДУЕТСЯ - Запускает все серверы автоматически
   - Запускает Backend (FastAPI) на http://localhost:8000
   - Запускает Frontend (React) на http://localhost:3000
   - Автоматически открывает браузер
   - Серверы работают в отдельных окнах

2. start-backend.bat
   ⚙️  Только Backend сервер
   - API доступен на http://localhost:8000
   - Документация API: http://localhost:8000/docs

3. start-frontend.bat
   🖥️  Только Frontend сервер
   - Интерфейс доступен на http://localhost:3000

4. stop-all.bat
   🛑 Останавливает все серверы
   - Завершает все процессы Python и Node.js


📌 КАК ИСПОЛЬЗОВАТЬ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ ПРОСТОЙ ЗАПУСК:
   1. Дважды кликните на start-all.bat
   2. Дождитесь запуска серверов (5-10 секунд)
   3. Браузер откроется автоматически
   4. Войдите: admin / admin123

⚠️  ОСТАНОВКА:
   - Закройте окна серверов (крестик или Ctrl+C)
   - ИЛИ запустите stop-all.bat

🔧 ЗАПУСК ПО ОТДЕЛЬНОСТИ:
   - Сначала запустите start-backend.bat
   - Потом запустите start-frontend.bat
   - Откройте http://localhost:3000 в браузере


🎯 СОЗДАНИЕ ЯРЛЫКОВ НА РАБОЧИЙ СТОЛ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Правой кнопкой по start-all.bat
2. Отправить → Рабочий стол (создать ярлык)
3. Переименуйте ярлык в "Запуск Снаб"
4. (Опционально) Измените иконку:
   - Правая кнопка → Свойства → Сменить значок

💡 УСТРАНЕНИЕ ПРОБЛЕМ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Порт занят:
   1. Запустите stop-all.bat
   2. Подождите 5 секунд
   3. Запустите start-all.bat снова

❌ Backend не запускается:
   - Проверьте наличие файла .venv\Scripts\python.exe
   - Убедитесь, что установлены зависимости

❌ Frontend не запускается:
   - Откройте папку frontend в терминале
   - Выполните: npm install
   - Попробуйте снова

❌ Браузер не открывается:
   - Откройте вручную http://localhost:3000
